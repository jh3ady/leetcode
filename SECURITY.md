# Security Policy

## Supported Versions

This repository tracks a continuously updated set of daily solutions. Only the `main` branch is supported.

## Reporting a Vulnerability

The code in this repository is educational (algorithmic puzzles). It is not intended for production use. If you still find an issue that could impact users who copy/paste a solution (for example an unsafe pattern), please report it privately.

Preferred channels:

1. GitHub Security Advisories: <https://github.com/jh3ady/leetcode/security/advisories/new>
2. Email: **admin@jdevelop.io**

Please do not open a public issue for undisclosed vulnerabilities. You will get an acknowledgement within 72 hours.
